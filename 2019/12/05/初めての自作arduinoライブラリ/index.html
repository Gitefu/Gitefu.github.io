初めての自作Arduinoライブラリ

Arduino, 電子工作

はじめに

って人向けに書いています。少々間違った言葉があっても悪しからず。

motorCtrlライブラリを作ろう

　今回はテストとしてTA7291Pのようなモータードライバを用いてモーターを-100~100[%]で回すためのライブラリ「motorCtrl」を作成します。やることは以下の2つだけ。
　・<.h>と<.cpp>を入れるためのディレクトリを作成
　・<.h>と<.cpp>の作成

クラスの概念

　作る前に少しクラスの話をしましょう．そんなに深く考えなくて良いです．クラスってのは“パーツの設計図” と思ってください．
　例えば今回の簡単なラジコンカーを作るとしましょう．

　図を見てください．ラジコンカーは軽い機能分けると「通信」と「車」ですね．さらに「車」を機能を細かくすると「モーターの制御」と「LEDランプの制御」ですかね(なんか車の目の部分ってLED？的なのついてますよね)．
　この場合はまずモーター制御クラスとLED制御クラスを作成します．次にこの2つのクラスを使って車クラスを作る訳です．さらに無線制御クラスも用意してしまえば，車クラスと無線制御クラスを組み合わせてラジコンカーのプログラムを作成できますね．
　「クラスというのはパーツの設計図」と最初に言いました．その意味がなんとなくでも掴めたらOKかなと思います．

準備

　以下の2つかな。
　・製作したArduinoのファイルが(デフォルトで)保存されるディレクトリはどこか把握する
　　(俺の場合はC:Documents/Arduinoでした。)
　・外部エディタを準備する
　　(win:VisualStudioCodeなど、mac:CotEditorなど)
　　(ファイルの中身を書き書きできればいい)

Arduinoライブラリの構造

　以下のファイルで構成されています。
　・ヘッダーファイル<.h>
　・C++ファイル<.cpp>
　・テキストファイル<keywords.txt>
 一番下のkeywords.txtの役割は「IDE上でこのファイル内で登録したキーワードの色を変える」というもので、必要不可欠ではありません。今回は<.h> と <.cpp> を作ります。

ディレクトリの作成

　Arduino/librariesの中に「motorCtrl」という名前のディレクトリを作ってください（ディレクトリ名は中に入れる<.h><.cpp>と同じにする必要があります。）

<.h>と<.cpp>

　ヘッダーファイル<.h>には宣言を．<.cpp>には中身を書いてください．
まあ一度作ればおそらくわかります．

motorCtrl.hの作成

　先ほど作成した「motorCtrl」ディレクトリのなかに「motorCtrl.h」を作りましょう．

 1#ifndef _MOTOR_CTRL_H_ //インクルードガード(呪文でOK)
 2  #define _MOTOR_CTRL_H_
 3  #include "arduino.h" //Arduinoのクラスを使うよってこと
 4  class motorCtrl{ //クラス名はファイル名と同じに！
 5    public: //クラスの外から使うモノはpublicに
 6      motorCtrl(unsigned int trueDirectionPinNum,unsigned int falseDirectionPinNum,unsigned int pwmPinNum); //インスタンス
 7      void initialize(void); //初期化
 8      void use(int powerPercent);  //回転(いい関数名浮かばなかった(Arduinoは予約語もあるのでよく分からん))
 9    private: //クラスの外から使わないモノはprivateに
10      unsigned int tPin;   //正転信号のピン
11      unsigned int fPin;   //逆転信号のピン
12      unsigned int pwmPin; //pwm信号のピン
13      unsigned int convert2PWMSignal(unsigned int powerPercent);//0~100→255
14  };
15#endif

中身はこんな感じ．インスタンスはあとで説明します．

motorCtrl.cppの作成

　先ほど作成した「motorCtrl」ディレクトリのなかに「motorCtrl.cpp」を作りましょう．

 1#include "motorCtrl.h" //さっき作ったファイルを呼び出し
 2/* インスタンスの中身*/
 3motorCtrl::motorCtrl(unsigned int trueDirectionPinNum,unsigned int falseDirectionPinNum,unsigned int pwmPinNum){
 4  tPin=trueDirectionPinNum; //privateな変数に保存
 5  fPin=falseDirectionPinNum; //privateな変数に保存
 6  pwmPin=pwmPinNum; //privateな変数に保存
 7}
 8
 9/* 初期化するための関数の中身*/
10void motorCtrl::initialize(void){
11  pinMode( tPin, OUTPUT);
12  pinMode( fPin, OUTPUT);
13  pinMode( pwmPin, OUTPUT);
14}
15
16/*  回転するための関数の中身
17    powerPercentは-100〜100[%]を想定
18*/
19void motorCtrl::use(int powerPercent){
20  if(powerPercent<-100||powerPercent>100)return; //範囲外なら実行しない
21  int direction = (powerPercent>0)?1:0;//0以上なら正転．0未満なら逆転
22  int powerPWMSignal;
23  if(direction==1){ //正転なら
24    powerPWMSignal = convert2PWMSignal((unsigned int)powerPercent); //%からPWM信号へ
25    digitalWrite(tPin,HIGH);
26    digitalWrite(fPin, LOW);
27    analogWrite(pwmPin,powerPWMSignal);
28  }else{//逆転なら
29    powerPWMSignal = convert2PWMSignal((unsigned int)(-1*powerPercent)); //%からPWM信号へ
30    digitalWrite(tPin, LOW);
31    digitalWrite(fPin,HIGH);
32    analogWrite(pwmPin,powerPWMSignal);
33  }
34}
35
36/*  %をPWMに変換する
37    powerPercentは0〜100[%]を想定
38    戻り値は0〜255を想定
39*/
40unsigned int motorCtrl::convert2PWMSignal(unsigned int powerPercent){
41  if(powerPercent>100)return 255;
42  return( (unsigned int)(((float)powerPercent * 255.0)/(float)(100.0)) );
43}

中身はこんな感じ．

ディレクトリの中身


　これで完成です．

テストプログラムの作成

　動作させるためにArduinoIDEで以下のプログラムを作成してコンパイルしてみてください．

 1#include "motorCtrl.h"
 2const unsigned int MOTOR_TRUE_DIRECTION_PIN =3;
 3const unsigned int MOTOR_FALSE_DIRECTION_PIN =4;
 4const unsigned int MOTOR_PWM_PIN =5;
 5motorCtrl motor(MOTOR_TRUE_DIRECTION_PIN,MOTOR_FALSE_DIRECTION_PIN,MOTOR_PWM_PIN); //インスタンス生成
 6void setup() {
 7  motor.initialize(); //初期化
 8}
 9
10void loop() {
11  motor.use(100);
12  delay(3000);
13  motor.use(50);
14  delay(3000);
15  motor.use(0);
16  delay(3000);
17  motor.use(-50);
18  delay(3000);
19  motor.use(-100);
20  delay(3000);
21  motor.use(0);
22  delay(3000);
23  /*
24   * 補足：publicのモノしかここからは呼び出せない！
25   */
26}

コンパイル通れば完成です！(筆者(私)は通ったので通るはずです．)

インスタンスって何？

　インスタンスはクラス(パーツの設計図)を基に作成されたパーツ(実体)と思ってください．<.h>に宣言を書き，<.cpp>にその中身を書きました．これで設計図ができた訳ですから，インスタンス(実体)を生成して使う訳ですね．
